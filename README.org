* Features
+ 32-bit (x86)
+ Modern Tetris gameplay (Piece holding, fast dropping, rotation in both directions)
+ Custom 8-bit RGB colour palette

* Dependencies
+ make
+ qemu
+ grub
+ mtools

* Running
** Unix-like
#+begin_src shell
make
make iso
qemu-system-i386 -cdrom boot.iso
#+end_src
